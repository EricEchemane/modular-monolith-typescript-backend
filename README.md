# Modular Monolith Express w/ Typescript, ORM, Jest

This repo will demostrate how to build a backend that is 100% testable, isolated to the real database, easy to mock on your own, and easy to scale. Uses dependency injection to make it easy to test and scale.

> Note: This isn't a microservice architecture, it's a monolith with modular architecture, but easier to scale and test.

## Recommended VSCode Extensions

- `orta.vscode-jest` (Jest with test coverage inline overlay so you can see how much of your code is covered by tests)
